IPL 2025 Exploratory Data Analysis (EDA) Project

Description
This repository contains a complete **Exploratory Data Analysis (EDA)** project on the **Indian Premier League (IPL) 2025 dataset**.  
The analysis focuses on understanding match trends, team performance, and season-level insights using **Python, Pandas, Matplotlib, and Seaborn**.

This project is designed to showcase my skills in:
- Data Cleaning  
- Data Visualization  
- Extracting Insights  
- Beginner-to-Intermediate Data Science Workflow  

Objectives
- Perform structured data cleaning and preprocessing  
- Analyze match outcomes and team-wise performance  
- Visualize trends using charts and graphs  
- Summarize key insights from the season  

 Tools & Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

 Dataset Information
The dataset contains match-level information such as:

- Teams playing  
- Match winners  
- Runs scored  
- Toss decisions  
- Player performance statistics  


 EDA Workflow Performed
 Data Loading and Inspection  
 Handling Missing Values  
 Data Cleaning and Preprocessing  
 Team Performance Analysis  
 Match Result Distribution  
 Visualization of Key Trends  
 Insights Extraction  

 
 Key Insights
- Top-performing teams were identified based on total wins  
- Match-winning trends such as chasing vs defending were analyzed  
- Run distributions and outcome patterns were visualized  
- Season-level competitive differences were observed
